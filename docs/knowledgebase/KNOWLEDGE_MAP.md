## 1. PRODUCT KNOWLEDGE

### 1.1 ColonAiQ Test Technology
- **Type:** Blood-based colorectal cancer screening test
- **Method:** Multi-gene methylation PCR detection
- **Biomarkers:** Septin9, BCAT1, IKZF1, BCAN, VAV3 (5-marker NMPA version); 6-marker version for India
- **Key docs:** IFU FINAL.docx, Workflow introduction, ColonAiQ Instructional Manual
- **Clinical performance:** ~86% sensitivity / 92% specificity (Gastroenterology 2021)

### 1.2 Clinical Applications
- **Primary:** First-line CRC screening (complement to colonoscopy, not replacement)
- **Secondary:** Minimal Residual Disease (MRD) monitoring post-surgery
- **Key docs:** Gastroenterology 2021 paper, JAMA Oncology 2023 MRD paper, Performance Evaluation study

### 1.3 Lab Workflow
- **Process:** Blood draw → sample processing → methylation PCR → result interpretation
- **Key docs:** Workflow introduction of ColonAiQ, Requirement for ColonAiQ assay(Auto)
